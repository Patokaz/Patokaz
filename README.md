## 👨‍💻 Sobre mim

Sou um Desenvolvedor Full-Stack com foco em **Front-End, UX/UI e experiências digitais modernas**.

Minha prioridade é desenvolver interfaces que não sejam apenas bonitas, mas também **intuitivas, rápidas e eficientes**.  
Acredito que um bom produto vai além do código — envolve entender o usuário, o problema e entregar a melhor solução.
